# 개인 프로젝트 [영화 검색 사이트]

### 🔗프로젝트 소개
---
이 프로젝트는 TMDb API를 활용하여 영화 정보를 검색할 수 있는 웹 애플리케이션입니다. <br>
사용자는 영화 제목을 검색할 수 있으며, 각 영화의 세부 정보를 모달 창을 통해 확인할 수 있습니다.


### 🕒프로젝트 기간
- - -
2024년 10월 15일 ~ 2024년 10월 23일


### ⚙기술 스택
---
`JAVASCRIPT`  `HTML`  `CSS` `The Movie Database API`



### 📌주요 기능
- - -
**메인 페이지**
+ 영화 포스터 카드
+ 영화 검색


**모달창**
+ 영화 상세 정보
+ 포스터 카드 클릭 -> 모달 오픈
  + X 누르면 창 닫힘
  + 모달 외부 클릭 창 닫힘
 

**검색**
+ 실시간 검색 기능
+ 정확한 값만 출력
